# syringe

Clojure code suitable for dev profile injection

## Usage

### Leiningen

####
profiles.clj dependencies
`[waffletower/syringe "0.1.0"]`

####
profiles.clj injections
`(require 'syringe.dose
          'syringe.midje)`
